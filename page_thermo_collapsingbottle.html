에탄올 연소 후 정수기 통 찌그러짐 시뮬레이션

정수기 통 내부를 이상기체처럼 단순화하여, 온도 T, 압력 P, 몰수 n, 부피 V의 변화를 시각화합니다.
대기 상태
온도 T 1.00
내부 압력 P 1.00 atm
기체 몰수 n 100%
통 부피 V 100%

단순화한 모델

P = nT / V

입구가 열려 있을 때 P가 커지면 내부 기체가 밖으로 빠져나가므로 n이 감소합니다.

입구를 막은 뒤에는 n이 고정되고, 냉각으로 T가 감소하면서 내부 압력이 낮아집니다.

그 결과 외부 대기압이 통을 안쪽으로 밀어 V가 감소합니다.

1. 에탄올 분무 후 대기 상태
2. 점화 → 연소 → 내부 온도 상승
3. 입구가 열려 있어 기체 유출
4. 입구를 막아 몰수 n 고정
5. 냉각 → 압력 감소 → 통 찌그러짐
